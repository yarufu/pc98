## このプロジェクトについて

ADV98 は、ChatGPT と Codex の支援を受けながら開発しているPC-98向けアドベンチャーゲームエンジンです。

コード生成や設計検討の多くを AI に手伝ってもらいながら、実際の実装・調整・画像制作・最終判断は作者が行っています。

**「AIでどこまでレトロゲーム開発ができるのか」**

という実験も兼ねて開発を続けています。

## オプション機能

### FM音源出力

KAJA氏によって開発された PMD（PC-98用 FM音源音楽ドライバ）を常駐すると、BGM機能が有効になります。

未常駐時でもゲーム本編は動作します。

### マウス操作

MS-DOS標準のマウスドライバを常駐すると、マウスカーソルは表示されませんが、左クリックで文字送りが可能になります。

未常駐でも動作します。  
その場合はキーボード操作のみになります。
