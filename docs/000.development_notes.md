# 第26話：保留：セーブ日時表示

日付：2026/6/12


## 背景

セーブデータに日時を保存し、
セーブ／ロード画面へ表示する案。

表示例：

01  96/06/12 20:24

## 状況

実装自体は可能。

DOS日時取得
- INT 21h AH=2Ah
- INT 21h AH=2Ch

で取得できることを確認済み。

## 保留理由

現在のセーブ画面レイアウトでは、

- 一列表示になる
- 文字が小さくなる
- 見栄えが微妙

などの問題がある。

## 将来案

案1：
セーブ枠を再設計して日時を1行表示

案2：
日時を2行表示

96/06/12
20:24

案3：
ASCIIフォントで日時のみ小さく表示

## 優先度

低

セーブ／ロード機能自体は完成しているため、
UI再設計時に再検討する。
