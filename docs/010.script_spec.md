# #bg #left #right #choice などの正式仕様の覚書

## 一覧


| 命令 | 用途 |
|---|---|
| `#bg` | 背景変更 |
| `#left` | 左立ち絵変更 |
| `#right` | 右立ち絵変更 |
| `#choice` | 2択選択肢 |
| `#label` | ジャンプ先ラベル |
| `#jump` | ラベルへ移動 |
| `#set` | フラグON |
| `#reset` | フラグOFF |
| `#if` | フラグONならジャンプ |
| `#ifnot` | フラグOFFならジャンプ |
| `#add` | 数値加算 |
| `#ifge` | 数値条件分岐 |
| `#pal` | パレット読み込み |
| `#bgm` | BGM読み込み＆再生 |
| `#bgmstart` | BGM再開 |
| `#bgmstop` | BGM停止 |
| `#bgmfade` | BGMフェードアウト |
