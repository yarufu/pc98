# #bg #left #right #choice などの正式仕様の覚書

## 一覧


| 命令           | 用途             |
| ------------ | ---------------- |
| `#bg`        | 背景変更           |
| `#bgwipe`    | 背景ワイプ表示      |
| `#left`      | 左立ち絵変更       |
| `#leftwipe`  | 左立ち絵ワイプ表示  |
| `#right`     | 右立ち絵変更       |
| `#rightwipe` | 右立ち絵ワイプ表示  |
| `#choice`    | 可変長選択肢開始   |
| `#endchoice` | 可変長選択肢終了   |
| `#label`     | ジャンプ先ラベル   |
| `#jump`      | ラベルへ移動     |
| `#call`      | サブルーチン呼び出し |
| `#return`    | 呼び出し元へ復帰 |
| `#set`       | フラグON        |
| `#reset`     | フラグOFF       |
| `#if`        | フラグONならジャンプ |
| `#ifnot`     | フラグOFFならジャンプ |
| `#setnum`    | 数値代入         |
| `#add`       | 数値加算         |
| `#ifeq`      | 数値一致条件分岐   |
| `#ifge`      | 数値以上条件分岐   |
| `#pal`       | パレット読み込み   |
| `#bgm`       | BGM読み込み＆再生 |
| `#bgmstart`  | BGM再開        |
| `#bgmstop`   | BGM停止        |
| `#bgmfade`   | BGMフェードアウト |

## FM効果音仕様

ADV98.EXE では、PMD の FM効果音機能を使用します。

FM効果音データは、実行ファイルと同じフォルダに配置した `SE.EFC` から読み込みます。

```
SE.EFC
```
