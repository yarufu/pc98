## FM効果音仕様

ADV98.EXE では、PMD の FM効果音機能を使用します。

FM効果音データは、実行ファイルと同じフォルダに配置した `SE.EFC` から読み込みます。

`#se <番号>` で0～127のFM効果音を再生します。引数が範囲外、数値以外、
または `SE.EFC` が読み込まれていない場合は、何もせずゲームを続行します。

```
#se 2
```

効果音番号 1 はシステム効果音として使用されています。

## 立ち絵インターレース表示

`#leftinterlace <キャラ番号> <表情>` と
`#rightinterlace <キャラ番号> <表情>` は、立ち絵SPRを一度読み込んだ後、
横方向の走査線を段階的に増やしながら表示します。

既存の `#left` / `#right` / `#leftwipe` / `#rightwipe` とは別命令です。

```
#leftinterlace character01 normal
#rightinterlace character02 happy
```
