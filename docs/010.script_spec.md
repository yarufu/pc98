## FM効果音仕様

ADV98.EXE では、PMD の FM効果音機能を使用します。

FM効果音データは、実行ファイルと同じフォルダに配置した `SE.EFC` から読み込みます。

`#se <番号>` で0～127のFM効果音を再生します。引数が範囲外、数値以外、
または `SE.EFC` が読み込まれていない場合は、何もせずゲームを続行します。

```
#se 2
```

効果音番号 1 はシステム効果音として使用されています。

## 背景表示

背景命令は、拡張子を含むG98ファイル名を直接指定します。
ファイル名はMS-DOSの8.3形式（ベース名8文字以内、拡張子3文字以内）とし、
ディレクトリや空白は使用しません。エンジンは`.G98`を自動付加しません。

`#bg <G98ファイル名>`は通常表示、
`#bginterlace <G98ファイル名>`は640×400全体をインターレース表示します。
背景の上側は偶数ラインを上から下へ、下側は奇数ラインを下から上へ、
1ラインおきに段階的に表示されます。背景名には通常のG98ファイルを指定します。

```
#bg CLASS.G98
#bginterlace CLASS.G98
```

引数なし、またはファイル読み込み失敗時は、黒画面に`BG LOAD NG`を表示して
ゲームを続行します。

演出後は、GameStateに保存されている現在の左右立ち絵を再描画します。
セーブ時は通常背景と同じくG98ファイル名だけを保存します。
ロード時は完成背景として復元され、インターレース演出は再実行されません。

## 立ち絵インターレース表示

立ち絵命令は、拡張子を含むSPRファイル名を直接指定します。
ファイル名はMS-DOSの8.3形式（ベース名8文字以内、拡張子3文字以内）とし、
ディレクトリは指定しません。エンジンは`.SPR`を自動付加しません。

`#leftinterlace <SPRファイル名>` と
`#rightinterlace <SPRファイル名>` は、立ち絵SPRファイルを1回openし、
32ライン固定バッファへ描画帯ごとに読み込みながら、横方向の走査線を段階的に増やして表示します。

通常表示の `#left` / `#right` とは別命令です。

```
#left C01_N.SPR
#right C02_H.SPR
#leftinterlace MIO_UNI.SPR
#rightinterlace MIO_EV1.SPR
#left none
#right none
```

`none`は立ち絵を消去します。従来のキャラクター番号と表情名の引数は廃止しました。
制服・私服・イベント専用画像などを、任意の8.3形式SPR名で指定できます。
