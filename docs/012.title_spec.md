# タイトル画面

ADV98.EXE 起動後、最初にタイトル画面を表示します。

タイトル画面では以下の項目を選択できます。

```txt
はじめから
ロード
終了
```

## はじめから

新しくゲームを開始します。

フラグ、数値フラグ、背景、立ち絵、スクリプト位置などの状態を初期化して、
`script.txt` の先頭から開始します。

## ロード

ロード画面を表示します。

セーブデータを選択すると、
保存時点の状態からゲームを再開します。

## 終了

ADV98.EXE を終了し、DOSへ戻ります。


# タイトル用ファイル

タイトル画面では、以下のファイルを使用します。

```txt
TITLE.G98
TITLE.PAL
TITLE.M
```

## TITLE.G98

タイトル画面用の背景画像です。

存在する場合、タイトル画面表示時に読み込まれます。

## TITLE.PAL

タイトル画面用のパレットファイルです。

存在する場合、タイトル画面表示時に読み込まれます。

## TITLE.M

タイトル画面用のBGMファイルです。

存在する場合、タイトル画面表示時に再生されます。

PMD.COM が常駐していない場合、BGM再生は行われません。


# タイトル画面へ戻るタイミング

以下の場合、タイトル画面へ戻ります。

```txt
script.txt の終端へ到達した場合
システムメニューで「タイトルへ戻る」を選んだ場合
```

`script.txt` の最後まで実行しても ADV98.EXE は終了せず、
タイトル画面へ戻ります。

タイトル画面へ戻ると、再度 `TITLE.G98`、`TITLE.PAL`、`TITLE.M` が適用されます。


# DOSへ戻るタイミング

以下の場合、ADV98.EXE を終了してDOSへ戻ります。

```txt
タイトル画面で「終了」を選んだ場合
右クリックシステムメニューで「ゲーム終了」を選んだ場合
```

`script.txt` 終端ではDOSへ戻りません。
