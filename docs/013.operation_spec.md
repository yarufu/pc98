# キーボード・マウス操作の覚書

# 操作方法

## キーボード

| キー            | 動作                          |
|--------         |--------                       |
| Enter           | 決定・メッセージ送り           |
| ↑↓←→            | 選択肢移動                    |
| テンキー8/2/4/6  | 選択肢移動                    |
| Esc              | システムメニュー              |
| テンキー0         | システムメニュー              |
| H                | メッセージウィンドウ表示/非表示 |

## マウス

| 操作       | 動作               |
|--------    |--------           |
| 左クリック | 決定・メッセージ送り |
| 右クリック | システムメニュー     |
