# スクリプトのサンプル

## 通常会話1

```
#pal adv.pal
#bgwipe class
#rightwipe character02 normal
#bgm kirakira.m

[健一]
ここ、静かだな

[]
誰もいない教室は、さっきまでの喧騒が嘘みたいに落ち着いている。そんな中、僕を呼び出した張本人は、いつもと変わらない様子でこちらを見ていた。

[美緒]
うん。今なら誰も来ないと思う

[健一]
それで、どうして僕を？

[美緒]
単刀直入に言うね。……犯人、知りたくない？

[]
いきなり何の話だろう。少し戸惑いながら聞き返す。

[健一]
犯人って？

[美緒]
期間限定のタルト。冷蔵庫に入れてたやつ、なくなってたでしょ？

[]
ああ、あの大事そうにしていたやつか。

[健一]
あれのことか

[美緒]
それ、私のなんだから

[]
視線が、少しだけ鋭くなる。

[健一]
僕は食べてないよ

[美緒]
でも、最後に冷蔵庫の前にいたの、健一でしょ？

[健一]
たまたま通りかかっただけだよ

[美緒]
じゃあ、違うって証明できる？

[]
どうしようかと迷う。ここで強く否定しても、信じてもらえない気がした。

[健一]
証明って言われても、どうすればいいのか

[美緒]
簡単だよ。一緒に犯人探し、手伝って

[]
やっぱり、そうなるよな……断れる雰囲気じゃない。

[健一]
でも、それって僕がやる必要あるのか？

[]
思わず口に出していた。少しだけ、納得がいかなかった。

[美緒]
そうだね。巻き込んでるのはわかってる

[]
さっきまでの強気な調子が、少しだけやわらぐ。

[美緒]
ごめん。でも、一人だと不安でさ

[]
その一言で、さっきの反発はすっと消えた。

[健一]
わかった。僕でよければ、手伝うよ

[美緒]
ありがとう

[]
小さく笑う美緒は、いつもより少しだけ柔らかく見えた。

[美緒]
じゃあまずは、アリバイ確認からだね

[]
この時はまだ、ただのいたずらだと思っていた。あんなことになるなんて、考えもしなかった。
```
