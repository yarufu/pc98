# ADV98 セーブ仕様

## 概要

ADV98 は `SAVE.DAT` にゲーム状態を保存する。

セーブ対象は主に以下。

* 現在のスクリプト位置
* 現在の背景
* 左立ち絵
* 右立ち絵
* BGMファイル名
* 通常フラグ
* 数値フラグ

---

## SaveData

```c
typedef struct {
    char magic[8];
    int version;
    GameState state;
    GameFlag flags[MAX_FLAGS];
} SaveData;
```

### magic

セーブデータ識別用。

例：

```txt
ADV98SAV
```

### version

セーブデータ形式のバージョン。

### state

現在のゲーム状態。

### flags

通常フラグ・数値フラグの保存領域。

---

## GameState

```c
typedef struct {
    char bg_name[32];
    int script_line;

    enum StandId left_stand;
    enum FaceId left_face;

    enum StandId right_stand;
    enum FaceId right_face;

    char bgm[64];
} GameState;
```

### bg_name

現在の背景名。

例：

```txt
class
gym
```

### script_line

現在のスクリプト行。

ロード時はこの位置から再開する。

### left_stand / left_face

左側立ち絵と表情。

### right_stand / right_face

右側立ち絵と表情。

### bgm

現在のBGMファイル名。

例：

```txt
kirakira.m
```

---

## GameFlag

```c
typedef struct {
    char name[32];
    int value;
} GameFlag;
```

通常フラグと数値フラグを共通で管理する。

例：

```txt
route_a = 1
money = 1000
love = 50
```
