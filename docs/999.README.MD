■ 概要

ADV98.EXE は、90年代のPC-98風アドベンチャーゲームを作成するためのサンプルゲームエンジンです。


■ 動作環境

[確認済み環境]
・PC-9821Ra43(PC-9801-86)
・PC-9821V13(PC-9801-118)

[OS]
・MS-DOS 6.2
・MS-DOS 5.0A-H


■ ファイル説明

ADV98.EXE  : 本体
script.txt : シナリオ
class.g98  : 背景画像
c01_n.spr  : キャラクター画像
adv.pal    : パレットファイル
readme.txt : このテキスト
PLAY.M     : BGMファイル（任意）
SE.EFC     : FM効果音データ（任意）


■ 実行方法

ADV98.ZIP を解凍後、フォルダごと PC-98 へコピーしてください。

その後、ADV98.EXE を実行してください。

■ システムメニュー

ゲーム中、
・Escキー
・テンキー0
・マウス右クリック
でシステムメニューを表示できます。


■ セーブとロード
セーブおよびロードはシステムメニューから実行できます。


■ マウス操作について

マウス操作を使用する場合は、MS-DOS用マウスドライバを事前に常駐してください。

例：

A:\>MOUSE
A:\>ADV98

マウスドライバが存在しない場合は、キーボードのみで操作できます。


■ 音楽機能について

BGMおよびFM効果音を使用する場合は、PMD.COM を事前に常駐してください。

例：

A:\>PMD
A:\>ADV98

PMD.COM が常駐していない場合は、

・BGMは再生されません
・FM効果音は再生されません

ゲーム本体はそのまま動作します。
