## 立ち絵描画

### 試したこと

| 候補 | 結果 |
|--------|--------|
| G98 fread回数 | 効いた |
| SPR fread回数 | ほぼ効かない |
| 背景復元 | 犯人ではない |
| SPR再描画 | 1回だけ |

### 調査結果

- graph98_draw_sprite_file_trans() は1回だけ実行されている
- ui_draw_stand() は複数回呼ばれるが STAND_NONE は即 return
- 背景描画より立ち絵描画の方が体感で遅い
- 現時点では大きな改善点は未発見

### 今後の候補
